# 🎨 Dirección Visual (ajustada al concepto)

**Paleta**

- Verde profundo #1a3a2a (madurez)
- Verde hoja #2f5e44 (crecimiento)
- Beige cálido (naturalidad)
- Blanco aireado (elegancia)
- Toque dorado suave para niveles premium

**Tipografía**

- Serif elegante para titulares (sensación orgánica)
- Sans limpia moderna para cuerpo
- Niveles con tipografía ligeramente distinta según categoría

**Estética**

- Mucho aire.
- Transiciones suaves.
- Sensación de “progreso natural”.
- Microanimaciones que simulan crecimiento (líneas que se expanden, barras que florecen).
